# Portfolio Website

Personal portfolio website for Arden Wang, showcasing projects and professional experience.

## Built With

- Next.js 14
- TypeScript
- Tailwind CSS
- Supabase
- EmailJS

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view in development mode.
